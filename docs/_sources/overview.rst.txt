Overview
========

Library with a collection of some more or less useful functions.


Installation
------------

.. highlight:: none

::

# pip install salmagundi


Dependencies
------------

- `ansictrls >= 0.1.0 <https://pypi.org/project/ansictrls/>`_
- `cryptography >= 2.4.2 <https://pypi.org/project/cryptography/>`_
